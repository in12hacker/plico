### Semantic FS
- **еҠҹиғҪ**: иҜӯд№үжҗңзҙў + BM25 fallback + vector index
- **ж–Үд»¶**: `src/fs/semantic_fs/`
- **Soul**: е…¬зҗҶ1 (AI-First)

### Knowledge Graph
- **еҠҹиғҪ**: еӣ жһңиҫ№ + SimilarTo + жқғйҮҚи·Ҝеҫ„
- **ж–Үд»¶**: `src/graph/`
- **Soul**: е…¬зҗҶ8 (еӣ жһңе…ҲдәҺе…іиҒ”)

### AI Kernel
- **еҠҹиғҪ**: Hook Registry + Circuit Breaker + Tool Dispatch
- **ж–Үд»¶**: `src/kernel/`
- **Soul**: е…¬зҗҶ5 (жңәеҲ¶дёҚжҳҜзӯ–з•Ҙ)

### Layered Memory
- **еҠҹиғҪ**: Ephemeral вҶ’ Working вҶ’ Long-Term вҶ’ Procedural
- **ж–Үд»¶**: `src/memory/layered/`
- **Soul**: е…¬зҗҶ4 (е…ұдә«е…ҲдәҺйҮҚеӨҚ)

### Agent Scheduler
- **еҠҹиғҪ**: Agent з”ҹе‘Ҫе‘Ёжңҹ + иө„жәҗй…Қйўқ + е№¶еҸ‘и°ғеәҰ
- **ж–Үд»¶**: `src/scheduler/`
- **Soul**: е…¬зҗҶ6 (иҮӘдё»е…ҲдәҺиў«еҠЁ)

### Intent Executor (N21-25)

- **ж–Үд»¶**: `src/kernel/ops/intent*.rs`
- **Soul**: е…¬зҗҶ2 (ж„Ҹеӣҫе…ҲдәҺж“ҚдҪң), е…¬зҗҶ7 (дё»еҠЁе…ҲдәҺиў«еҠЁ), е…¬зҗҶ9 (и¶Ҡз”Ёи¶ҠеҘҪ)
